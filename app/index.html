The goal of this page

  • Please, note that I do it in my freetime as a hobby, so be patient.
  • This page contains learning materials in different topics.
  • First of all to help in psychological/mental improvement and well-being.
  • Language learning contents are also under development and publication, mainly in swedish.

Technical information

  • The content packages supports the SCORM 1.2 standard.
  • They can be downloaded and used in an LMS system.
  • I didn't want to use a complete LMS here, because the goal of the frame is to create materials, and not to track the learning progress.
  • The word lists can be listen as you learn words in a language school.
